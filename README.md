# Number Guessing Game (Python)

A simple command-line game where the computer chooses a random number between 1 and 100, and you try to guess it.  

## How to Play
- Run `number_guessing_game.py`
- Enter a number between 1 and 100
- The game tells you if your guess is too high or too low
- Keep guessing until you win
- Type `0` to quit anytime

## Features
- Tracks number of guesses
- Simple, beginner-friendly Python project
